# 💼 Salon CRM Dashboard UI

A modern **CRM Dashboard UI design** created for a beauty salon to manage customers, appointments, staff, and daily operations efficiently.

## 📌 Project Overview

This project is a **web-based CRM dashboard** tailored for a salon business (Glow Beauty Salon).
It helps streamline daily workflows like:

**Customer → Appointment → Service → Follow-up → Revenue Tracking**

The goal is to design a clean and user-friendly interface that improves productivity for small salon teams.

## 🎯 Features

* 📅 Appointment Management (daily schedule tracking)
* 👩‍💼 Customer Management (new & regular clients)
* 🔄 Lead Pipeline (New → Booked → Regular → VIP)
* 👩‍🔧 Staff Status (Available / Busy)
* 💰 Revenue Overview (daily earnings)
* 🔍 Search & Filter options
